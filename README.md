# Piña Hub - Centralizador de Recursos y Aplicaciones Web

Piña Hub es un entorno web dinámico diseñado para centralizar, organizar y previsualizar una colección de aplicaciones interactivas ligeras y recursos multimedia en una sola interfaz unificada. 

El objetivo principal del proyecto es optimizar el acceso a herramientas web externas mediante el uso de contenedores incrustados, evitando la apertura de múltiples pestañas y mejorando la eficiencia en entornos de red locales.

## ⚡ Características Principales
* **Carga en Contenedor Único:** Integración de recursos mediante componentes `iFrame` dinámicos para mantener la navegación en una sola pestaña.
* **Interfaz Fluida:** Diseño responsivo basado en CSS Grid optimizado para cargas rápidas de red.
* **Aislamiento de Entorno:** Scripts optimizados para pausar procesos en segundo plano cuando se cierra un recurso.

## 🛠️ Tecnologías Utilizadas
* **HTML5 Semántico:** Estructuración de tarjetas de recursos.
* **CSS3 Custom Properties:** Estética oscura adaptativa.
* **Vanilla JavaScript:** Control de flujos de inyección de URLs y gestión del scroll.
